3. Этап - Регламентирование

Разрабатываем нормативную документацию
  • Структуру управления предприятием c распределением функций;
  • Должностные инструкции с полномочиями и зоной ответственности;
  • Систему мотивации и оценки вовлеченности сотрудника в процесс;
  • Технологические инструкции (регламенты) выполнения операций.
